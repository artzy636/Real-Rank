# 🚀 Real Rank

Real Rank is a data-driven system that estimates the practical difficulty of LeetCode problems using:

* Official LeetCode difficulty
* Topic complexity
* Acceptance rate

The goal is to identify problems that may be easier or harder than their official LeetCode rating.

## Features

✅ Analyze 3559 LeetCode problems

✅ Search any LeetCode problem

✅ Difficulty score breakdown

✅ Difficulty mismatch detection

✅ Top 10 hardest problems

✅ Top 10 easiest problems

✅ Score distribution analytics
